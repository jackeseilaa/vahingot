iCloud & Cloud Sync

☁️ Vie iCloud:iin (Backup)

Vie kaikki tiedot iCloud Driveen tai muuhun pilvitallennukseen.

Ohjeet:
1. Klikkaa "Vie varmuuskopio"
2. Tiedosto ladataan: vahinkohallinta_backup_PPKK.json
3. Avaa Files-app
4. Vedä tiedosto → iCloud Drive / Vahinkohallinta
5. Nyt synkassa kaikilla laitteillasi! ✓

📥 Tuo iCloud:sta (Palauta)

Palauta tiedot joista laitteesta tahansa.

Ohjeet:
1. Avaa Files-app
2. Etsi vahinkohallinta_backup_PPKK.json
3. Jaa → Tämä sivu (tai vedä tiedosto)
4. Tuonti onnistuu automaattisesti ✓

🔄 Manuaalinen Sync

Synkronoi kahden laitteen välillä.

Esim. Mac → iPad:
1. Mac: Klikkaa "Vie varmuuskopio"
2. Lähetä tiedosto esim. sähköpostiin
3. iPad: Lataa sähköpostista
4. Klikkaa "Tuo varmuuskopio"
5. ✓ iPad:lla kaikki Mac:in data!
🚀 Tulossa: Automaattinen synkkaus iCloud:iin (ei manuaalista vientia tarvita)

Nyt: Vie/tuo manuaalisesti
Tulevaisuus: Auto-sync taustalla
